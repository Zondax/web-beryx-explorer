# Bloque

## Introducción

Esta página proporciona una vista detallada de un bloque específico en la cadena de bloques. Puedes acceder a esta página buscando un hash de bloque específico, haciendo clic en el ejemplo de búsqueda de bloque en la página principal o desde la página de tipset.

Encuentra un ejemplo de bloque [aquí](https://beryx.io/fil/mainnet/block-cid/bafy2bzaceb2udtsfnbz6viwq3mscy4msh4tcr5lhy7fscweovpsk4ksnl4sgu).

## ¿Qué contiene la página de Bloque?

La página de Bloque se divide en:

1. **Resumen del Bloque**: Esta sección muestra información clave sobre el bloque, como su ID de bloque, altura del tipset, CID del tipset, minero, timestamp y número de transacciones.
2. **Tabla de Transacciones**: Esta tabla enumera todas las transacciones incluidas en el bloque. Para cada transacción, puedes ver su estado, hash, emisor, receptor, método, cantidad y gas utilizado.

## Navegando por la página de Bloque

Puedes navegar por la página de Bloque utilizando las pestañas. En la parte superior, se encuentra la cadena y la red del bloque y su identificador.
