# Block

## Introduction

This page provides a detailed view of a specific block in the blockchain. You can access this page by searching a specific block hash, clicking on the block search example in the home page or from the tipset page.

Find a block example [here](https://beryx.io/fil/mainnet/block-cid/bafy2bzaceb2udtsfnbz6viwq3mscy4msh4tcr5lhy7fscweovpsk4ksnl4sgu).

## What does the Block page contain?

The Block page is divided into:

1. **Block Overview**: This section displays key information about the block such as its block ID, tipset height, tipset CID, miner, timestamp and number of transactions.
2. **Transactions Table**: This table lists all the transactions included in the block. For each transaction, you can see its status, hash, sender, receiver, method, amount, and gas used.

## Navigating the Block page

You can navigate the Block page using the tabs. On top, there is the chain and network of the block and its identifier.
